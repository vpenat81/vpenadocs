    GRUB2 (Grand Unified Bootloader v2) Bootloader that loads and transfers control to the Linux kernel /boot/grub2/grub.cfg
    PXE (Preboot Execution Environment) Boots system from a network server Booting diskless workstations using PXE server
    USB Boot Boots system from a USB flash drive Installing Linux from /dev/sdb1
    ISO Boot Boots system from an ISO image Booting from ubuntu.iso using virtual machine or USB
