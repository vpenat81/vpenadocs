    üîç Scanning and Tools

    Tool/Term Short Definition Example
    Vulnerability Management The ongoing process of identifying, assessing, prioritizing, and remediating security weaknesses. Regularly scanning systems to find and fix software bugs.
    CVE Common Vulnerabilities and Exposures; unique ID for vulnerabilities. CVE-2023-12345 identifying a specific software flaw.
    CVSS Common Vulnerability Scoring System; rates severity of vulnerabilities. CVSS score 9.8 indicating a critical vulnerability.
    CWE Common Weakness Enumeration; categorizes types of software weaknesses. CWE-79 refers to Cross-Site Scripting (XSS).
    OWASP Open Web Application Security Project; provides web app security guidelines. OWASP Top 10 listing critical web security risks.

    üîÑ Change Management

    Term Short Definition Example
    Change Any addition, modification, or removal of anything that could affect IT services. Upgrading the company‚Äôs email server software.
    Problem Underlying cause of one or more incidents affecting services. Repeated server crashes traced back to faulty software.
    Incident Unplanned interruption or reduction in quality of an IT service. Network outage causing loss of connectivity for users.
    Emergency Change Management Fast-tracked change process to address critical issues urgently. Patching a critical security vulnerability immediately.
    RCA (Root Cause Analysis) Process to identify the fundamental cause of a problem or incident. Investigating why a database failure occurred and fixing it.

    Term Short Definition Example
    Offboarding App Process of decommissioning an application from active use. Retiring an old HR system and migrating users to a new one.
