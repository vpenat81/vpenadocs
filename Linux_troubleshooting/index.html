    💡 Linux Troubleshooting

    📀 1.0 Storage issues

    ⚡ High Latency

    Issue Description Example
    Input/output (I/O) wait Time CPU waits for I/O to complete High iowait shown in top

    📉 Low Throughput

    Issue Description Example
    Low throughput System processes less data per time Slow file copy speed using dd or rsync

    🔢 Input/output Operations Per Second (IOPS) Scenarios

    Issue Description Example
    Low IOPS Low number of I/O operations/sec Low IOPS reported in iostat output

    🗃️ Capacity Issues

    Issue Description Example
    Low disk space Insufficient free storage Disk full warning from df -h
    Inode exhaustion No more inodes available df -i shows 100% inode usage

    🛠 Filesystem Issues

    Issue Description Example
    Corruption Filesystem data corruption fsck reports errors
    Mismatch Filesystem metadata mismatch Mount fails due to superblock mismatch

    ⚙️ I/O Scheduler

    Concept Description Example
    I/O scheduler Kernel subsystem managing I/O cat /sys/block/sda/queue/scheduler

    💾 Device Issues

    Device/Issue Description Example
    Non-volatile memory express (NVMe) High-speed storage interface nvme list
    Solid-state drive (SSD) Fast storage device SSD info via lsblk or smartctl
    SSD trim TRIM command to optimize SSDs fstrim /
    RAID Redundant disk array RAID status in cat /proc/mdstat
    LVM Logical Volume Manager Logical volumes shown by lvdisplay
    I/O errors Errors in I/O operations Errors in dmesg or smartctl -a

    ⚠️ Mount Option Problems

    Issue Description Example
    Mount option problems Incorrect mount parameters Use noatime for performance or read-only ro mount errors

    🖧 2.0 network resource issues

    🌐 Network Configuration Issues

    Issue Description Example
    Subnet Incorrect subnet settings Misconfigured subnet mask causing no connectivity
    Routing Incorrect route paths Missing route in ip route causing unreachable network

    🔥 Firewall Issues

    Issue Description Example
    Firewall rules Incorrect or blocking rules iptables -L blocking required ports

    ⚠️ Interface Errors

    Issue Description Example
    Dropped packets Packets dropped on interface ifconfig or ip -s link showing dropped packets
    Collisions Network packet collisions High collision count on ifconfig output
    Link status Interface link up/down status ethtool eth0 shows link is down

    📶 Bandwidth Limitations

    Issue Description Example
    High latency Delay in data transmission Ping times >100ms on local network

    🌍 Name Resolution Issues

    Issue Description Example
    DNS Issues Failure to resolve domain names nslookup google.com fails or times out

    🧪 Testing Remote Systems

    Tool Description Example
    Nmap Network scanner and port discovery nmap -sS 192.168.1.1 to scan open ports
    openssl s_client Test SSL/TLS connections openssl s_client -connect google.com:443 to check SSL handshake

    🖥️ 3.0 CPU and memory issues

    ⚙️ Runaway Processes

    Concept Description Example
    Runaway Processes Processes consuming excessive CPU Use top or htop to identify a 100% CPU task

    🧟 Zombie Processes

    Concept Description Example
    Zombie Processes Defunct processes waiting for parent ps aux | grep Z shows <defunct> status

    🔥 High CPU Utilization

    Concept Description Example
    High CPU Utilization When CPU is consistently maxed out top or vmstat showing >90% usage

    📊 High Load Average

    Concept Description Example
    High Load Average More processes waiting than CPUs available uptime shows load average: 12.0 on a 4-core system

    🏃 High Run Queues

    Concept Description Example
    High Run Queues Too many tasks waiting for CPU vmstat 1 shows high number under "r" column

    ⏱️ CPU Times

    Type Description Example
    user Time spent on user processes top, sar -u
    system Time spent on kernel processes top, iostat
    idle Time CPU is doing nothing vmstat, top
    iowait Waiting for I/O operations iostat -c
    steal Time stolen by hypervisor (VMs) top, mpstat

    🎚️ CPU Process Priorities

    Command Description Example
    nice Set initial priority of process nice -n 10 command
    renice Change priority of running process renice -n -5 -p 1234

    🧠 Memory Exhaustion

    Concept Description Example
    Free vs File Cache Memory Cached memory used by the system free -h, look under "buff/cache"

    🚫 Out of Memory (OOM)

    Concept Description Example
    OOM System runs out of memory Check /var/log/messages or dmesg for OOM killer
    Memory leaks Process consumes increasing memory Use top, smem, or valgrind
    Process killer Kernel kills process to free memory dmesg | grep -i 'killed process'

    💽 Swapping

    Concept Description Example
    Swapping Moving data from RAM to disk swap space vmstat, free -h

    🖥️ Hardware Info Commands

    Command Description Example
    lscpu Shows CPU architecture and specs lscpu
    lsmem Shows memory ranges and size lsmem
    /proc/cpuinfo CPU details from virtual file system cat /proc/cpuinfo | grep "model name"
    /proc/meminfo Memory details from virtual file system cat /proc/meminfo | grep MemTotal

    🕵️‍♂️ 4.0 User access and file permissions

    🔐 User Login Issues

    Issue Description Example
    User Login Issues Problems during user authentication Check auth.log, run who, last, or faillock --user USERNAME

    📁 User File Access Issues

    Issue Description Example
    Group User not in the correct group for file access groups username / usermod -aG group user
    Context SELinux/AppArmor preventing access ls -Z file.txt / chcon, restorecon
    Permission Incorrect file/directory permissions ls -l file.txt / chmod 755 file.txt
    ACL Extended file permissions restricting access getfacl file.txt / setfacl -m u:john:r file.txt
    Attribute Special attributes like immutable set on file lsattr file.txt / chattr -i file.txt
    Policy/non-policy Security modules denying access (SELinux, etc.) Check audit.log, use semanage, getsebool

    🔑 Password Issues

    Issue Description Example
    Password Issues Weak, expired, or locked user passwords passwd user / chage -l user / faillock -u user

    ⚙️ Privilege Elevation

    Method Description Example
    sudo Temporarily execute commands as another user sudo apt update
    su Switch to another user account su - username
    pkexec Graphical or policy-based privilege escalation pkexec gedit
    visudo Safely edit sudoers file sudo visudo

    📉 Quota Issues

    Issue Description Example
    Quota Issues Users exceeding disk space or inode limits quota -v / repquota /home / edquota user

    ❌ 5.0 Common problems with a Linux system

    🧩 Unit Files

    Concept Description Example
    Service A unit of type service, controls a process /etc/systemd/system/nginx.service
    Networking services Services related to network configuration NetworkManager.service, systemd-networkd.service
    ExecStart Command to start the service ExecStart=/usr/sbin/nginx
    ExecStop Command to stop the service ExecStop=/bin/kill -WINCH ${MAINPID}
    Before/After Order control for unit start sequence After=network.target
    Type Specifies how the service starts Type=simple, forking, oneshot
    User Runs service as specific user User=nginx
    Requires/Wants Declares dependencies Requires=network.target

    ⏰ Timer Unit Files

    Concept Description Example
    OnCalendar Specifies calendar-based scheduling OnCalendar=daily
    OnBootSec Delay after boot to activate service OnBootSec=5min
    Unit The service unit the timer activates Unit=backup.service
    Time expressions Schedule format OnCalendar=Mon *-*-* 02:00:00

    🗂️ Mount Unit Files

    Concept Description Example
    Naming conventions Based on mount path /mnt/data → mnt-data.mount
    What What device or path to mount What=/dev/sdb1
    Where Mount point Where=/mnt/data
    Type Filesystem type Type=ext4
    Options Mount options Options=defaults,noatime

    🎯 Target Unit Files

    Concept Description Example
    Default The default system target default.target → graphical.target
    Multiuser Multi-user text-based target multi-user.target
    Network-online Waits for network to be up network-online.target
    Graphical GUI session target graphical.target

    🚨 Common Problems

    Problem Description Example
    Name resolution failure DNS or hostname lookup issues systemd-resolve --status, check /etc/resolv.conf
    Application crash App fails to start or crashes during run Check logs via journalctl -u app.service
    Time-zone configuration Incorrect timezone settings Use timedatectl set-timezone America/New_York
    Boot issues Errors during system boot process journalctl -b, systemctl list-jobs
    Journal issues Missing or unreadable system logs Check journalctl, systemd-journald service status
    Services not starting on time Dependencies or incorrect ordering in units Ensure proper After= or Requires= settings in unit files
